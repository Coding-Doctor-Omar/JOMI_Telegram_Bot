# JOMI TELEGRAM BOT
![QR Code](images/JOMI_BOT_QR.jpeg)
Telegram URL: https://t.me/jomi_downloader_bot/

## What This Bot Does?
This bot allows medical students to access any JOMI surgical video for FREE!

## How to Use?
Send the bot the link for the JOMI video page, and the bot will unlock the video for you and send you new links to access the video in qualities that range from 224p to 1080p, according to you preference.

The bot will also generate for you an srt file for the English subtitles and send it to you on the fly.

## Available Commands
/start => Sends a welcome message.

/how_to_use => Teaches you how to use the bot.

/source_code => Sends you the link to this repository to check the bot's source code on GitHub.
